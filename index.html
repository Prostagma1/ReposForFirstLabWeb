Страница для 1.1 лабораторной работы

HTML (HyperText Markup Language) – это язык разметки, используемый для создания веб-страниц. Он позволяет описывать структуру документа и определять, какие элементы будут отображаться на странице и как они будут выглядеть. HTML является одним из основных языков веб-разработки и используется практически всеми веб-сайтами в мире.

Основными элементами HTML являются теги (tags), которые определяют структуру документа. Каждый тег начинается с символа "<" и заканчивается символом ">". Теги могут иметь атрибуты, которые определяют дополнительные свойства элемента, такие как размер, цвет, ссылки и т.д.

С помощью HTML можно создавать разнообразные элементы веб-страниц, такие как заголовки, параграфы, списки, таблицы, изображения, ссылки и другие элементы. Кроме того, HTML позволяет встраивать другие элементы веб-страниц, такие как видео, аудио и другие мультимедийные элементы. HTML является важным инструментом для создания функциональных и привлекательных веб-сайтов.
